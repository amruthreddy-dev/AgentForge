# Code of Conduct

## Our Pledge

We are committed to making participation in AgentForge a respectful and harassment-free experience for everyone.

## Expected Behavior

* Be respectful
* Be constructive
* Be inclusive
* Welcome feedback

## Unacceptable Behavior

* Harassment
* Personal attacks
* Discrimination
* Abusive language

## Enforcement

Project maintainers may remove contributions or comments that violate this Code of Conduct.

## Scope

This Code of Conduct applies to all project spaces, issues, discussions, and pull requests.
